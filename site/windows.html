After download

Install, open, translate your first text.

The Windows version follows the same core flow: install the app, keep it in the tray, select text, press a hotkey.

1 1 minute to first translation Unzip the app, open JellyTranslate, test in Notepad. ⌄
  1. Unzip the download

    Extract JellyTranslate to a normal folder, for example Downloads or Documents.

  2. Open JellyTranslate

    Run JellyTranslate.exe. Windows may ask you to confirm opening a new app.

  3. Try it in Notepad

    Select “Hello world”, then press Ctrl+Alt+T.
