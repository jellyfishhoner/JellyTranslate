Why JellyTranslate?

Because translating text on macOS is still slower than it should be.

No browser tabs. No copy-paste loops. No switching context. Just select text, press a hotkey, translate or replace.

1

Select text

Highlight text in TextEdit, Notes, Safari, Chrome, Telegram, or another Mac app.

2

Press shortcut

Open the popup or translate and replace selected text directly with your chosen hotkeys.

3

Get translation

Choose the target language in the popup, copy the result, or replace the original text.

For early testers

Install, enable permissions, test in TextEdit first.

macOS may ask for Accessibility and Input Monitoring permissions so JellyTranslate can read selected text, use clipboard fallback, and replace text where possible.

This test version is not signed yet. If macOS says the app is damaged, macOS is blocking the downloaded file, not reporting a broken app.

1 1 minute to first translation Open the test version safely, grant permission, test in TextEdit. ⌄
  1. Drag to Applications

    Open the ZIP, then drag JellyTranslate.app into the Applications folder.

  2. If macOS blocks it

    Open Terminal and paste this command once. It only removes the download safety label from JellyTranslate.

  3. Grant permission

    Enable Accessibility for JellyTranslate, then test “Hello world” in TextEdit.

Updating later

Quit JellyTranslate, download the newest build, then replace the old app in Applications.

Privacy & Security

Accessibility must be enabled for selected-text reading.
