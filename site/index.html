Native macOS menu bar translator

Translate selected text anywhere on Mac.

Select text, press a shortcut, and JellyTranslate shows a compact floating translation popup. Built for quick daily use, not for getting in your way.

Early macOS test build. MyMemory works without an API key for MVP testing.

1

Select text

Highlight text in TextEdit, Notes, Safari, Chrome, Telegram, or another Mac app.

2

Press shortcut

Open the popup or translate and replace selected text directly with your chosen hotkeys.

3

Get translation

Choose the target language in the popup, copy the result, or replace the original text.

Small by design

A Mac utility that quietly does one thing well.

JellyTranslate lives in the menu bar, uses native SwiftUI and AppKit, and keeps the core experience simple: selected text in, translation out.
