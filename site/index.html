For early testers

Install, enable permissions, test in TextEdit first.

macOS may ask for Accessibility and Input Monitoring permissions so JellyTranslate can read selected text, use clipboard fallback, and replace text where possible.

This is an unsigned alpha build. If macOS says the app is damaged, the app is being blocked by Gatekeeper quarantine, not by a broken download.

1 1 minute to first translation Open the alpha safely, grant permission, test in TextEdit. ⌄
  1. Move the app

    Unzip JellyTranslate and move it to Applications.

  2. If macOS blocks it

    Run this once for the unsigned alpha build.

    xattr -dr com.apple.quarantine /Applications/JellyTranslate.app
  3. Grant permission

    Enable Accessibility for JellyTranslate, then test “Hello world” in TextEdit.
